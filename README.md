# 🎨 Color Palette Picker

A simple and interactive **Color Palette Picker** built with **React**.  
Users can choose a color from a set of color buttons, preview the selected color, and reset their selection.

## ✨ Features

- 🎨 Choose from different colors
- ✅ Shows a checkmark on the selected color
- 🖼️ Displays a preview box of the selected color
- 📝 Shows the selected color name
- 🔄 Reset button to clear the selection
- 📱 Simple and responsive design

## 🛠️ Technologies Used

- React
- JavaScript 
- CSS
- Vite

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone your-repository-link
```

### 2. Go to the project folder

```bash
cd color-palette-picker
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the development server

```bash
npm run dev
```

The project will open in your browser.

## 📖 How It Works

- The selected color is stored using React's `useState`.
- Each color button updates the selected color when clicked.
- The preview component displays the selected color.
- Pressing the reset button clears the selected color and returns to the empty state.
